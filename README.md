# <span style='color:#008800'>Parsley command line option parsing library</span>

Parsley is a C++ library that may be used to parse command line options.
Parsley sprinkles a little flavor on your parsing.
Parsley was inspired, in part at least, by the python click module.

Parsley was developed on and for Linux, however it should be readily
buildable on other operating systems.

Parley documenation is generated by doxygen, located in the html/ directory.
